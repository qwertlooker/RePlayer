# AGENTS.md

## Product Goal
Build a portable Windows English-learning audio player for local MP3 study.

## Must Have
- MP3 playback
- SRT/LRC subtitle display
- subtitle click-to-seek
- single-line repeat
- auto pause between lines
- record user's voice for current line
- compare original line audio and recorded audio

## Constraints
- Windows 10/11
- portable, no installer
- no admin rights
- offline-first
- free/open dependencies preferred

## Non-goals
- no login
- no cloud
- no AI scoring
- no speed control
- no global hotkeys
- no playlist management in v1

## Engineering Priorities
1. Stability
2. Simplicity
3. Clear UI state
4. Modular subtitle/audio/recording code
