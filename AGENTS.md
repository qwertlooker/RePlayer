# AGENTS.md

## 产品目标
构建一个轻量、免安装、面向英语学习的 Windows 本地播放器。

第一版专注“可用”，后续逐步向 IonPlayer 的能力演进。

---

## v1 核心目标
实现一个最小可用的英语精听工具。

---

## 用户核心流程
1. 打开本地 MP3
2. 加载 SRT / LRC 字幕
3. 查看同步字幕
4. 点击句子跳转播放
5. 单句复读
6. 句间自动暂停
7. 对当前句录音
8. 对比原音与录音

---

## 平台与技术约束

### 平台
- Windows 10 / 11
- 原生桌面程序（Win32）
- 绿色版（无需安装）
- 不需要管理员权限

### 技术栈
- C++17
- Win32 API（界面）
- Media Foundation（音频播放）
- WASAPI（录音）
- CMake 构建

### 依赖原则
- 不使用 Qt / Electron / .NET / Python
- 尽量零依赖或极少依赖
- 控制最终体积

---

## v1 必须功能

### 播放
- 打开 MP3
- 播放 / 暂停 / 停止
- 支持 seek
- 显示时间

### 字幕
- 支持 SRT
- 支持 LRC
- 字幕列表显示
- 当前句高亮
- 点击跳转

### 学习功能
- 单句复读
- 句间自动暂停（可配置秒数）

### 录音
- 当前句录音
- 保存为 WAV
- 可回放录音

### 对比
- 播放当前句原音片段
- 播放当前句录音

---

## 工程原则

### 1. 简单优先
- 不做复杂抽象
- 先实现功能闭环

### 2. 模块清晰
必须拆分：
- audio（播放）
- subtitles（字幕）
- recording（录音）
- ui（界面）
- state（状态）

### 3. 状态集中
- 使用统一 AppState
- 禁止状态分散

### 4. 时间逻辑可控
- 播放时间必须可预测
- 避免漂移

### 5. 错误可见
- 所有错误必须提示用户
- 日志记录详细信息

---

## 状态规则

### 播放模式
- Normal（普通）
- RepeatOne（单句复读）
- AutoPause（句间暂停）

优先级：
RepeatOne > AutoPause

---

### 录音规则
- 每句一个录音
- 允许覆盖
- 文件绑定句子

---

## UI 原则

必须明确显示：
- 当前句
- 当前模式
- 是否正在录音

界面要求：
- 简洁
- 优雅
- 无隐藏状态
- 操作直接

---

## 日志

输出路径：
logs/app.log

必须记录：
- 文件加载
- 字幕解析
- 播放异常
- 录音状态
- 模式切换

---

## 交付要求

- 单目录绿色版
- 双击即可运行
- 不依赖额外运行时
